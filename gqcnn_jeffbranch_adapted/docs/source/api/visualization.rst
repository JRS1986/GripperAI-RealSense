Visualization
=============

Visualizer
~~~~~~~~~~
A visualization tool specific to GQCNN visualization needs. Different from visualizers in the BerkeleyAutomation Perception module. GQCNN specific 
function include visualizing grasps and much more.

.. autoclass:: gqcnn.Visualizer

GQCNNPredictionVisualizer
~~~~~~~~~~~~~~~~~~~~~~~~~
A tool for visualizing GQCNN predictions on a specific dataset. Allows for the visualization of true positives, true negatives,
false positives, false negatives and their corresponding grasps and other metrics such as precision and recall.

.. autoclass:: gqcnn.GQCNNPredictionVisualizer
